Acerca del Proyecto Kiosko de Apple

Este proyecto es un kiosko virtual interactivo que permite explorar la historia de Apple de forma visual y táctil.

Construido con Python y Django, y ejecutado en un miniPC Chuwi con Ubuntu, este sistema está pensado para mostrar información obtenida desde distintas webs mediante scraping.
